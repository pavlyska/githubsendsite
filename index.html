GitHub Send Commit

Получайте уведомления о новых коммитах в публичных репозиториях GitHub — быстро и без авторизации.

→ Запустить бота

✨ Возможности

  • 🔔 Мгновенные уведомления о новых коммитах
  • 🌍 Поддержка языков: русский / english
  • 📥 Управление подписками через кнопки
  • ⏱ Проверка каждые 10 секунд

❗ Важно!

Бот не требует входа в GitHub.

Мы используем собственные токены для получения коммитов — никогда не отправляйте свои логины, пароли или персональные токены!

▶️ Как начать?

  1. Перейдите в @githubsend_bot
  2. Нажмите «Подписаться»
  3. Введите репозиторий в формате: владелец/название (например: torvalds/linux)
  4. Готово! Новые коммиты будут приходить вам в Telegram.

Следите за open-source — безопасно, удобно и без спама.
